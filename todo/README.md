## TODO list in go

objective : practice go concepts


commands:

todo - general

add - add string to a file or some sort of database for register

rm - remove based on id of a file or some sort of database that got registers 

list - show tasks in a way that get the string and id of the task 

check - target task as being done 

version - show current version of todo list cli 


task model:

description - string - description of the task 
id - string - id of the task 
isDone - boolean - verify if the tasks is done or not
